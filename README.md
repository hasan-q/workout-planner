# Gym Workout Planner
A full stack gym workout planner built with Spring Boot, PostgreSQL, and React to help users plan workouts, track exercise progress, and visualize progress over time.
<img width="1280" height="800" alt="Dashboard of the Workout Planner" src="https://github.com/user-attachments/assets/86e529e4-bde8-49ec-8f07-c1e13817cefc" />

## Tech Stack

**Frontend:** React.js, Axios, Recharts, CSS
**Backend:** Java Spring Boot, Spring Security, JWT Authentication
**Database:** PostgreSQL
**Dev Tools:** Postman, Docker, VS Code

# Running the Program

First, make sure git is installed on your system, Then, clone this repository by running `git clone https://github.com/hasan-q/workout-planner.git`.
